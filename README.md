# gulp

A standard Gulpfile, written in ES6, with tasks for JavaScript, Vendor JavaScript, SCSS and SVG data URI encoding. Configurable using an external `gulp-config.js` file, thus you can customise the Gulp tasks to suit your requirements, without needing to touch the `Gulpfile.js` file itself.
